Nodes development stages​

  1. coming soon: Just an idea.

  2. under development: The node is under development.

  3. closed alpha: Testing with a closed-set of developers.

  4. testing: Anyone can use the node but It's still not stable.

  5. public: Stable enough but still facing some issues or more features are coming soon.

  6. stable: The node is stable for all use-cases.
